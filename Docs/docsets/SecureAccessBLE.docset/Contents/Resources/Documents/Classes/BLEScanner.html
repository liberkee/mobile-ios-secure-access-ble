BLEScanner

public class BLEScanner: NSObject, DataTransfer, CBCentralManagerDelegate, CBPeripheralDelegate

BLEScanner implaments the secure BLE session between mobile devices and SID. The communication manager can only send / receive messages over a secure BLE connection, i.e. a valid session context must exist.
